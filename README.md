# head_first_design_patterns

书籍《Head First设计模式》的读书笔记

使用JavaScript的超集typescript语言实现[相关代码案例](https://github.com/bethrobson/Head-First-Design-Patterns)

# 安装编译依赖库
```sh
  npm install
```

# 构建命令
```sh
  webpack
```
